# フロントエンド開発環境（テンプレート）

## Directory

```
├── dist                // 公開用ファイル
├── src                 // 編集ファイル格納ディレクトリ
│   ├── sass
│   └── js
├── gulpfile.js
├── package.json
└── webpack.config.js
```

## Getting Started

```bash
$ npm i
```

## Task Runner

開発用

```bash
$ gulp
```

本番用

```bash
$ gulp --type production
```
